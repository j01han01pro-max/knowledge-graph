# Naming Standards

## Classes

    URI = Ontology Namespace + ClassName

Example:

    Namespace: https://semantic-data.jlh.com/kg/ipm/ipmg#
    ClassName: Revolution
    URI = https://semantic-data.jlh.com/kg/ipm/ipmg#Revolution

### Class Naming Guidelines

#### Format

    No spaces in or around value


    Case-sensitive
    Examples/Notes: DataType is not the same LUI value as Datatype


    Subsequent words start with upper case with no spaces (use CamelCase notation)
    Examples/Notes: BillOfMaterialDocument, SalesOrganization


    Use all upper case for acronyms
    Examples/Notes: LOB instead of Lob


    First word starts with upper case
    Examples/Notes: Batch

#### Content

    Only alpha or numeric characters, no symbols, special characters or punctuation
    Examples/Notes: Use characters A-Z, a-z, 0-9


    Avoid character formation (use plain ASCII format, avoid accents)
    Examples/Notes: Telephone or Telefono instead of Tel√©fono


    Use explicit and concise names
    Examples/Notes: WallOfEsophagus or EsophagusWall instead of WallOfTheEsophagus


    Use singular rather than plural
    Examples/Notes: Service instead of Services
